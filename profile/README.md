### What we're building

**APICircle Studio** — an open-source API client where every action is a command, collections live in Git, and AI manages your workspace through MCP.

```
> create post /api/users in /Auth named "Create User"
> run all in /Payment --env=staging
> branch switch feature/v2-payments
> diff results
```

### Core principles

- **Commands over clicks** — terminal-native UX, keyboard-first
- **Git over cloud sync** — branch, PR, diff your API collections
- **AI-native** — MCP server lets Claude, Cursor, and agents manage your APIs
- **Composable** — multi-repo workspaces, separate concerns with access control
- **Open source** — MIT licensed, community-driven
